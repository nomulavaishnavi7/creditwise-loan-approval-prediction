# creditwise-loan-approval-prediction
Machine Learning project for predicting loan approval using classification algorithms.


## Overview
This project predicts whether a loan application is likely to be approved based on applicant financial and demographic information. The objective is to assist financial institutions in making data-driven lending decisions.

## Dataset Features
- Applicant Income
- Credit Score
- Debt-to-Income Ratio (DTI)
- Employment Status
- Other applicant-related attributes

## Machine Learning Workflow
1. Data Cleaning and Preprocessing
2. Missing Value Handling using SimpleImputer
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. One-Hot Encoding
6. Feature Scaling using StandardScaler
7. Model Training and Evaluation

## Models Implemented
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Gaussian Naive Bayes

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix
